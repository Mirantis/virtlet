# Virtlet 101 workshop

In this workshop you will learn how to:

* Deploy Virtlet on [kubeadm-dind-cluster](https://github.com/kubernetes-sigs/kubeadm-dind-cluster)

Kubernetes Version: v1.11  
Virtlet Version: v1.2.0
